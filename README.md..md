**E-Commerce Sales \& Customer Behavior Analyzer**



**Project Overview**



This project analyzes e-commerce sales performance, customer behavior and revenue trends using Python, SQL and Power BI.



**Technologies Used**



\* Python

\* SQL

\* Power BI

\* Pandas

\* Excel/CSV



**Features**



\* Total Revenue Analysis

\* Product Performance Analysis

\* City Wise Sales Analysis

\* Payment Method Analysis

\* Monthly Revenue Trend Visualization



**Dashboard Insights**



\* Total Revenue KPI

\* Top Selling Products

\* Revenue by City

\* Payment Method Distribution

\* Revenue Trend Analysis
